attr 与 prop attr和prop的不同： attr：访问的是元素的属性。 当属性没有被设置时候，.attr()方法将返回undefined。 src的属性 alt属性 把src属性的值设置为title属性的值。 prop：访问的是元素对应DOM对象的属性 若要检索和更改DOM属性,比如元素的checked, selected, 或 disabled状态，请使用.prop()方法。 attr 概念：获取匹配的元素集合中的第一个元素的属性的值 或 设置每一个匹配元素的一个或多个属性。 attr的语法：(谱绕破儿踢死)(内容，属性) attr(name|properties|key,val|fn) name:这里第一个值是属性名； 作用：这里的name属性返回文档中所有图像的src属性值 例子： $("images").attr("src"); properties ：作为属性的“名/值对”对象 作用：为所有图像设置src和alt属性 例子： $("img").attr({ src: "test.jpg", alt: "Test Image" }); key,value：属性名，属性值 作用：为所有图像设置src属性 例子： key,function(index,attr):1：第一个是属性名，2:返回属性值的函数,第一个参数为当前元素的索引值，第二个参数为原先的属性值。 作用：把src属性的值设置为title属性的值。 例子： $("img").attr("title", function() { return this.src }); prop (性能) 概念：获取匹配的元素集中第一个元素的属性（property）值或设置每一个匹配元素的一个或多个属性。 prop的语法： prop(name|properties|key,value|fn) name:这个是属性名称 作用：选中复选框为true，没选中为false 例子： $("input[type='checkbox']").prop("checked"); properties|：作为属性的“名/值对”对象 作用：禁用页面上的所有复选框。 例子： $("input[type='checkbox']").prop({ disabled: true }); key,value:属性名称，属性值 作用：禁用和选中所有页面上的复选框。 例子： $("input[type='checkbox']").prop("disabled", false); $("input[type='checkbox']").prop("checked", true); key,function(index, attr)：1:属性名称。 2:返回属性值的函数,第一个参数为当前元素的索引值，第二个参数为原先的属性值。 作用：通过函数来设置所有页面上的复选框被选中。 例子： $("input[type='checkbox']").prop("checked", function( i, val ) { return !val; }); HTML5 data-* 自定义属性 创作者定义的属性。 HTML 文档的创作者可以定义他们自己的属性。 必须以 "data-" 开头。 1.设置方法 A.直接在标签内书写，格式如下
